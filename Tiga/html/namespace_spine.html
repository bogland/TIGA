◆ MixPose

enum Spine.MixPose
strong

Controls how a timeline is mixed with the setup or current pose.

참고
Timeline.Apply(Skeleton, float, float, ExposedList, float, MixPose, MixDirection)
열거형 멤버
Setup 

The timeline value is mixed with the setup pose (the current pose is not used).

Current 

The timeline value is mixed with the current pose. The setup pose is used as the timeline value before the first key, except for timelines which perform instant transitions, such as DrawOrderTimeline or AttachmentTimeline.

CurrentLayered 

The timeline value is mixed with the current pose. No change is made before the first key (the current pose is kept until the first key).

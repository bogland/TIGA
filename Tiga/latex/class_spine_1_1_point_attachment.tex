\doxysubsection{상세한 설명}
An attachment which is a single point and a rotation. This can be used to spawn projectiles, particles, etc. A bone can be used in similar ways, but a \mbox{\hyperlink{class_spine_1_1_point_attachment}{Point\+Attachment}} is slightly less expensive to compute and can be hidden, shown, and placed in a skin. 

See \href{http://esotericsoftware.com/spine-point-attachments}{\texttt{ Point Attachments}} in the \mbox{\hyperlink{namespace_spine}{Spine}} User Guide.
